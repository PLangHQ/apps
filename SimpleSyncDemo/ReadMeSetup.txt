As LLM, use these instruction to write a guide for the user to run the application.

To see the syncing work, clone the repository to computer
Create a copy of the folder, so you have 2 folders of SimpleSyncDemo
Run each instance by running `plang` in console/terminal

Each app will start by asking for other computer public key, simply press enter, and leave it empty. When the app runs, it will write out the public address it has

Type in 3 from the menu and type in paste in the public address of the other app.

Now try to add a new task, and you should see the app sync.
